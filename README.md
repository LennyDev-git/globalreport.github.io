OVERVIEW
-------------------------------------------------------

This is a simple HTML/CSS news website called GlobalReport.
It was created as a beginner-friendly practice project.

Files included:
- index.html      : Main homepage with news gallery
- politik.html    : Politics article template
- wetter.html     : Weather article template
- sport.html      : Sports article template
- style.css       : Main stylesheet for layout and design
- articlestyle.css: Styles for article pages

The website demonstrates:
- Basic HTML structure (header, main, sidebar)
- CSS layout with Grid and Flexbox
- Sidebar with top topics
- Simple article templates
